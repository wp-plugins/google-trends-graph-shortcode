Generate Google Trends graphs on you WordPress website using shortcode.

== Description ==

**Google Trends** is an extremely simple plugin that'll generate Google Trends graph(s) on your website for up to 5 search terms on a single chart. 

Features include:

* Specify the height and width of the graph.
* Compare up to 5 search terms in the same query on the same chart.
* Compare search terms globally or limit the query to a specific country.

== Installation ==

**To install the plugin manually:**

1. Extract the contents of the archive (zip file)
2. Upload the google-trends-graph-shortcode folder to your '/wp-content/plugins' folder
3. Activate the plugin through the Plugins section in your WordPress admin panel.

**Upload via the WordPress administration panel:**

1. Click on "Plugins" in the left panel, then click on "Add new".
2. You should now see the Install Plugins page. Click on "Upload".
3. Click on Browse and select your "google-trends-graph-shortcode.zip" file.
4. Click on "Install now", activate it and you're done!

**What will the plugin do?**
